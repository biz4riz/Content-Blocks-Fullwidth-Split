Content Blocks Fullwidth Split

Split content contained with fullwidth background color/image.

Classes

Inset classes add padding, dictated by the number of columns set within the class, on either side of the element. inset[breakpoint][columns]

Pull classes add margins, dictated by the number of columns set within the class, on the left side of the element. push[breakpoint][columns]

Push classes add margins, dictated by the number of columns set within the class, on the left side of the element. push[breakpoint][columns]
